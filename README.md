Applepie is a convenient sdk for ios project.

## Features

- [x] MVVM
- [x] Network
- [x] Cache
- [x] Location
- [x] Schema
- [x] Extensions


## Requirements

- iOS 9.0+

## Installation

### CocoaPods

#### Swift 4.0 - Swift 4.2
```ruby
pod 'Applepie'
```
